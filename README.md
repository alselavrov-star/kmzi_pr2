Репозиторий содержит:
прорграммный код rsa.py реализует работу криптоалгоритма RSA в режимах генерации ключей, зашифрования и расшифрования;
прорграммный код rsa_details.py аналогичен работе кода rsa.py, но с подробным выводом информации по всем операциям;
прорграммный код attack_rsa.py реализкет атаку на RSA методом факторизации Ферма.
